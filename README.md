# Attendance Management System

A Spring Boot (Java 17) based Attendance Management System for managing students, teachers, subjects, and attendance records.

## Features
- Student Management
- Teacher Management
- Subject Management
- Attendance Tracking
- Excel Import (.xlsx)
- Export Attendance to Excel & PDF

## Technologies Used
- Java 17
- Spring Boot
- MySQL
- Maven
- Apache POI
- OpenPDF

## Run Project

```bash
mvn spring-boot:run
